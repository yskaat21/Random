# Clopos Ana Sayfa

Bu proje Next.js, React ve TypeScript kullanarak Clopos restoran yönetim sistemi için oluşturulmuş bir ana sayfadır.

## Başlarken

Geliştirme sunucusunu çalıştırın:

```bash
npm run dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açarak sonucu görebilirsiniz.

Sayfayı düzenlemek için `src/app/page.tsx` dosyasını değiştirebilirsiniz.

## Özellikler

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
